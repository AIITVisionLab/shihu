# Modbus 寄存器映射与轮询流程

## 从站映射

- 从站1 地址：`1`
  - `40001` -> 光照 ADC
- 从站2 地址：`2`
  - `40011` -> 温度
  - `40012` -> 湿度
- 从站3 地址：`3`
  - `40021` -> MQ2 ppm

## 主站参数

- 串口：`USART2`
- 引脚：`PD5/PD6 + PB8(DE/RE)`
- 串口格式：`9600 8E1`
- 单事务超时：`1000ms`
- 失败重试：`2` 次
- 请求间隔：`10ms`
- 轮询周期：`1000ms`

## 每轮顺序

1. 读从站1 `40001`
2. 读从站2 `40011`、`40012`
3. 读从站3 `40021`
4. 生成一份采集快照并入队上报

## 异常处理

- 超时：标记对应从站 `online=0`
- CRC 错误：标记 `lastError=CRC`
- 协议异常：标记 `lastError=PROTOCOL`
- Modbus 异常响应：标记 `lastError=EXCEPTION`
- 串口硬件错误：标记 `lastError=PORT`
