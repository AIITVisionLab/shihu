# RK3568 直连网络方案

## 拓扑

- STM32F429 只有一个网口，直接连接 RK3568 的一个网口。
- RK3568 的另一个网口或其他上行接口连接外部网络。
- F429 不直接感知外部网络，只和 RK3568 的直连接口通信。

## 默认地址规划

- F429：`172.18.8.240/24`
- RK3568 直连接口：`172.18.8.18/24`
- F429 网关：`172.18.8.18`

## 说明

- F429 不会因为直连 RK3568 就“断网”，它只是工作在一个单独子网中。
- 如果 RK3568 需要继续把数据发往外网，必须由 RK3568 负责路由、转发或应用层转送。
- 本工程不实现 NAT、桥接或双网口管理逻辑。
