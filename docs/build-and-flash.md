# 构建与烧录

## 构建方式

- 使用 `project/CMakeLists.txt`。
- 目标产物：`template.elf`、`template.hex`、`template.bin`。

## 本轮约束

- 本轮没有由代理执行编译。
- 需要你手动执行 `CMake Configure/Build`。
- 若有报错，把完整报错发回，我按报错继续修正。

## 烧录前检查

- 确认 F429 的 RS485 接线为 `PD5/PD6 + PB8`。
- 确认 F429 与 RK3568 直连口地址规划一致。
- 确认 3 个 F103 从站寄存器映射保持原工程定义。

## 联调建议顺序

1. 先只接调试串口，确认系统能启动。
2. 再接 RS485，从站逐个验证。
3. 最后接网线，与 RK3568 验证 HTTP 上报。
